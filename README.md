# Nightscout Ottai / Syai CGM Uploader

Скрипт на Python для периодической отправки данных о глюкозе от фолловера Ottai или Syai в Nightscout. Поддерживает многопользовательский режим.

> Протестировано на российских датчиках Ottai (белая коробка) и Syai.

## ⚠️ Важное изменение

Изменён формат переменных окружения. При запуске программа выводит список доступных мастеров для фолловера. Для настройки конкретного мастера префикс переменной определяется по следующей цепочке приоритетов:

1. **Email** — часть до знака `@`. Для `test@example.com`:
   - `NS_URL__test`
   - `NS_SECRET__test`

2. **userName** — если у мастера нет email (например, при особом типе приглашения). Для `userName=user049395`:
   - `NS_URL__user049395`
   - `NS_SECRET__user049395`

3. **fromUserId** — числовой ID пользователя как крайний fallback. Для `fromUserId=20394353`:
   - `NS_URL__20394353`
   - `NS_SECRET__20394353`

При запуске скрипт выводит подходящий ключ для каждого ненастроенного мастера.

## ⚙️ Конфигурация

Скрипт использует следующие переменные окружения:

| Переменная | Описание | Пример | Обязательно |
|------------|----------|--------|-------------|
| **OTTAI_TOKEN** | Ottai API JWT токен | `eyJhbGciOiJIUzUxMiJ9...` | Да |
| **HOURS_AGO** | Глубина загрузки данных при каждом цикле (в часах) | `5` | Да |
| **NS_URL\_\_\<ключ\>** | URL Nightscout для конкретного мастера | `NS_URL__test=https://j12345678.nightscout-jino.ru` | Да* |
| **NS_SECRET\_\_\<ключ\>** | API Secret Nightscout для конкретного мастера | `NS_SECRET__test=H4uSur24Bkwu` | Да* |
| **OTTAI_BASE_URL** | Сервер API. Для Syai: `https://ru.syai.com`. Для Ottai оставьте пустым (по умолчанию `https://seas.ottai.com`). | `https://ru.syai.com` | Нет |
| **OTTAI_CUSTOMER_ID** | ID фолловера — fallback, если API не возвращает список пользователей | `20251217` | Нет |
| **DISABLE_SSL_VERIFY** | Отключает проверку SSL-сертификатов | `true` | Нет |

\* Нужна хотя бы одна пара `NS_URL__<ключ>` / `NS_SECRET__<ключ>`. Ключ подбирается автоматически при запуске.

## 🔑 Получение API JWT токена (Ottai / Syai)

### 1. Настройка мастера и фолловера
- Мастер выбирает «поделиться» в приложении Ottai или Syai и вводит адрес почты фолловера
- **Важно:** Нельзя быть одновременно мастером и фолловером. Адреса почт должны отличаться!

### 2. Установка приложений
- Установите **Ottai Share** (или **Syai**) на телефон фолловера и зарегистрируйтесь
- Установите приложение для захвата пакетов:
  - **PCAPdroid** для Android
  - **Захват HTTP-трафика** для iOS
- Настройте сертификат согласно инструкции приложения для захвата пакетов

### 3. Получение токена
- Запустите захват трафика и откройте приложение фолловера
- В заголовках запросов найдите `authorization:"..."` — это ваш `OTTAI_TOKEN`
- Также в заголовках найдите `customer_id` — это значение для переменной `OTTAI_CUSTOMER_ID`

> **Примечание:** Токен выдаётся на полгода и должен оставаться неизменным. При переустановке приложения или повторном входе/выходе процедуру нужно повторить.

## 🐳 Запуск с помощью Docker

### Сборка контейнера
```bash
docker build -t ottai-nightscout-uploader .
```

#### Запуск через docker compose

1. Создайте файл `.env` в корне репозитория:
```env
OTTAI_TOKEN=ваш_токен
HOURS_AGO=5

# Для Syai:
OTTAI_BASE_URL=https://ru.syai.com
# Для Ottai оставьте OTTAI_BASE_URL пустым или не указывайте

# Nightscout для каждого мастера (ключ подскажет скрипт при первом запуске):
NS_URL__mariatar96=https://ваш_адрес.nightscout-jino.ru
NS_SECRET__mariatar96=ваш_секрет
```

2. Запустите из корня репозитория:
```
docker compose up -d
```

> При первом запуске без настроенных NS-переменных скрипт выведет список мастеров с нужными ключами.

### 📱 Запуск на телефоне через Termux (Android)

1. Скачайте и установите [termux](https://play.google.com/store/apps/details?id=com.termux) 

2. Установка пакетов
```
pkg up -y && pkg install openssl python3 git openssl-static openssl-tool
```

3. Клонирование репозитория
```
git clone https://github.com/pmakhotkin/ottai-nightscout-uploader.git
```

4. Установка зависимостей
```
cd ottai-nightscout-uploader/
pip install -r requirements.txt
```
5. Настройка переменных окружения

```
nano .env
```
Внесите свои данные по образцу:
```env
OTTAI_TOKEN=ваш_токен
HOURS_AGO=5
OTTAI_BASE_URL=https://ru.syai.com  # только для Syai
NS_URL__ключ=https://ваш_адрес.nightscout-jino.ru
NS_SECRET__ключ=ваш_секрет
```
Сохраните файл (Ctrl+O, Enter, Ctrl+X).

6. Загрузка переменных окружения

```
set -a && source .env && set +a
```

7. Запуск скрипта

```
python -u main.py
```
